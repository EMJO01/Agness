# Scentra Perfumery 🌸

Client landing page for Scentra Perfumery — a luxury fragrance brand.

## What it is
A premium landing page built for a perfume brand client, designed to communicate luxury, showcase products, and drive customer enquiries through WhatsApp.

## Features
- Rich visual design — luxury feel, dark aesthetic
- Product showcase section
- WhatsApp order/enquiry button
- Fully mobile-responsive
- Fast loading

## Tech Stack
- HTML · CSS · JavaScript

## Purpose
Client project — showcased in the EminnTech portfolio as an example of luxury brand web design for Nigerian businesses.
